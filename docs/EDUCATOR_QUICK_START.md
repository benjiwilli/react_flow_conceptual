## ğŸ¯ Your First Custom Workflow

### Building a Vocabulary Lesson

1. **Add Student Profile Node**
   - Drag "Student Profile" from Node Library
   - Configure ELPA level (1-5)
   - Set native language

2. **Add Vocabulary Builder Node**
   - Drag "Vocabulary Builder" to canvas
   - Connect from Student Profile
   - Enter target vocabulary words

3. **Add L1 Bridge Node**
   - Drag "L1 Bridge" to canvas
   - Connect from Vocabulary Builder
   - This translates to student's home language!

4. **Add Visual Support Node**
   - Drag "Visual Support" to canvas
   - Connect from L1 Bridge
   - AI generates helpful images

5. **Run & Review**
   - Click "Run" button
   - Watch your personalized lesson come to life!

## ğŸ’¡ Tips for Success

### Best Practices

1. **Start Simple**
   - Begin with 3-4 nodes
   - Test frequently as you build
   - Add complexity gradually

2. **Know Your Students**
   - Set accurate ELPA levels
   - Configure native languages
   - Adjust grade levels appropriately

3. **Use L1 Bridges Strategically**
   - Add after new vocabulary
   - Use for complex instructions
   - Celebrate home languages!

4. **Monitor and Adjust**
   - Watch the classroom dashboard
   - Respond to teacher alerts
   - Modify workflows based on results

### Common Questions

**Q: Can I save my workflows?**
A: Yes! Click the "Save" button. Workflows are saved to your browser in demo mode.

**Q: How do I share with students?**
A: Copy the workflow URL or use the classroom session feature to assign to students.

**Q: What if a student is stuck?**
A: The classroom dashboard shows alerts. You can provide hints or adjust the workflow difficulty.

**Q: Can I edit while running?**
A: Stop the execution first, make changes, then run again.
